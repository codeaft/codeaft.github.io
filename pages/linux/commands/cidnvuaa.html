touch and cat commands to create the files
codeaft@codeaft:~$ ls
a.out  KW.class  Codeaft.java  Desktop    Downloads	Pictures  Templates  Weekdays
codeaft.c   codeaft.csv	 codeaft.py	  Documents  Music	Public	  Videos

codeaft@codeaft:~$ touch Ubuntu.txt Fedora.txt
codeaft@codeaft:~$ ls
a.out	  codeaft.csv   Desktop    Fedora.txt  Public      Videos
codeaft.c	  Codeaft.java  Documents  Music	  Templates   Weekdays
KW.class  codeaft.py    Downloads  Pictures	  Ubuntu.txt

codeaft@codeaft:~$ cat > KaliLinux
^C
codeaft@codeaft:~$ ls
a.out	  codeaft.csv   Desktop    Fedora.txt  Pictures   Ubuntu.txt
codeaft.c	  Codeaft.java  Documents  KaliLinux   Public     Videos
KW.class  codeaft.py    Downloads  Music	  Templates  Weekdays

codeaft@codeaft:~$ file Fedora.txt codeaft.c codeaft.csv
Fedora.txt: empty
codeaft.c:       C source, ASCII text
codeaft.csv:     CSV text
rm command to remove the file(s)
codeaft@codeaft:~$ rm KaliLinux Weekdays
codeaft@codeaft:~$ ls
a.out	  codeaft.csv   Desktop    Fedora.txt  Public      Videos
codeaft.c	  Codeaft.java  Documents  Music	  Templates
KW.class  codeaft.py    Downloads  Pictures	  Ubuntu.txt
rm command with a regular expression to remove multiple files of the same extension
codeaft@codeaft:~$ rm *.txt
