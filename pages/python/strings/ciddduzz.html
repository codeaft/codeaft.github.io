Strings Concatenation
>>> s1="Koding"
>>> s2="window"
>>> s1+s1
'KodingKoding'

>>> s1+s2
'Codeaft'

>>> s1*5
'KodingKodingKodingKodingKoding'

>>> s1,s2
('Koding', 'window')
Escape Characters
>>> codeaft="Koding\nwindow"
>>> codeaft
'Koding\nwindow'

>>> codeaft="Koding\nwindow"
>>> print(codeaft)
Koding
window
 
>>> codeaft=r"Koding\nwindow"
>>> print(codeaft)
Koding\nwindow
 
>>> print('\"Codeaft\"')
"Codeaft"

>>> codeaft="Koding\twindow"
>>> print(codeaft)
Koding	window
